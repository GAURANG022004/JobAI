# JobAI Backend

Production-ready backend foundation built using:

- Java 21
- Spring Boot 3.x
- Maven

## Current Features

- Spring Boot starter
- Layered architecture
- Configuration using YAML
- Logging
- Ready for future modules

## Planned Modules

- Resume Generation
- Cover Letter Generation
- ATS Analysis
- Job Analysis
- Chrome Extension
- Local AI Integration (Ollama)
- Form Autofill
- Application Tracking
